Headers and Paragraphs

Headers in HTML are used describe the current page with a short description for the main purposes of providing pertanent information for search engines. That way if a person did a search on google, the page's header could be used to see if this particular page may satisfy the user's search. Headers can also be seen in the page's tab on your web browser

Paragraphs are insertted into the body in an HTML file. A paragraph is preceded by "p" with the open and close carrots around it and then closed with a "/p", also with carrots. Using paragraphs is an effective way to separate text for a more readable body.
