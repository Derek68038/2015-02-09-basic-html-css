Head and Body

The head part of an HTML file is where the header would be inputted. You would insert the header between pairs of the "title" syntax, with carrots around each.

The body part of an HTML file is then where you would insert your paragraphs. Other types of information which could go in the body are lists and hyperlinks, which you've already seen examples of in this document.
